# frontend

## 写在最前

在前端方面, 我是一个很新的新手, CSS以及布局方面我想很难提高了. 为了之后的同学维护
简单, 我就不用`Vue`这样的框架了.

采用Bootstrap+jQuery, 不管写成什么样, 这的确是我的极限了.

采用`webpack`打包, 目前使用的是4.2版本, 这边已经可以正常使用了.

如果你对`node`不熟悉, 推荐使用`WebStorm`, 打开项目安装依赖即可.

## 运行方式

```bash
> npm install
> npm run dev   # 开启调试服务
> npm run build # 生成文件
```
