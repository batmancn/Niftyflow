## 使用apidoc生成文档

> 使用方式参见[apidoc](http://apidocjs.com)

```
> npm install apidoc -g
> apidoc -i ./controler/ -o ./doc/apidoc # 生成的文档保存在doc/apidoc中
```


所有文档采用`Markdown`编写, 使用`pandoc`转换为`tex`文件后编译, 模板会在之后附上.
